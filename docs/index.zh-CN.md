# InputGPT 
 InputGPT 一个[开源的](https://github.com/linexjlin/inputGPT)跨平台的软件 让你任何可以打字的地方，直接调用LLM。 默认的续写模式， 让你写任何文档都可以像使用 IDE 一样有补全能力。
